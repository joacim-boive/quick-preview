FAQ

Short answers.

What video formats are supported?

QuickPreview supports the same video formats that native macOS preview support handles by default. If a file is not supported there, install an additional codec/plugin package and it should work in both places.

Recommended resources: Apple Pro Video Formats and QuickLookVideo plugin project.

Does QuickPreview replace a full editor?

No. The product is positioned as a fast review and looping tool, not a full editing timeline or media management suite.

Can it focus on one specific moment repeatedly?

Yes. The product flow includes full-loop playback as well as start and end markers for replaying a selected segment.

Can I manage saved review points in one place?

Yes. QuickPreview includes a bookmark manager for reopening saved moments, working across clips, and separating protected bookmarks from standard ones when needed.

How do the Up and Down arrows work now?

Up and Down move through bookmarks while the bookmark manager is open, so you can switch clips or moments without first focusing the bookmarks table. Use Shift + Up/Down when you want frame stepping instead.

Why does a bookmark stop opening the right clip?

Bookmarks reference the original file path. If the source media is moved, renamed, or removed, that bookmark reference will no longer match and needs to be updated.

What if I lose my paranoid mode password?

The paranoid mode password cannot be recovered. To start over, reset the app state by reinstalling and setting the app up again. Your original media files remain intact.

How does QuickPreview handle my media files?

QuickPreview does not move or duplicate your media. It stores references to file paths for playback and bookmarks. If a file is moved, renamed, or deleted, related bookmarks need to be updated to match the new location.

Do bookmarks and settings sync across Macs?

Not automatically in this version. Bookmarks and clip state are stored locally on each Mac, so use your normal machine backup workflow if you want recovery or migration coverage.

What happens when protected media is locked again?

Protected bookmarks are hidden until you unlock again. If paranoid mode is enabled, unlocking requires macOS authentication plus the paranoid password step.

What is the fastest codec troubleshooting flow?

First test the file in native macOS preview support. If it fails there, install codec support, then reopen QuickPreview and test again.

Privacy FAQ

Private by default.

QuickPreview is designed as a lightweight local app. It does not collect personal user profiles, upload your media, or build a cloud account around your usage.

Does QuickPreview store information about me?

QuickPreview does not collect or store personal information about its users on our side. There is no user account, no cloud profile, and no media upload pipeline built into the app.

What does the app store locally on my Mac?

To make the app useful, QuickPreview stores local app data such as bookmarks, clip state, and protected-bookmark settings on your own Mac. That information stays on your device unless you back it up or sync your machine using your own system tools.

Do you upload my videos or bookmark data anywhere?

No. QuickPreview works with local file references and local app state. Your videos are not uploaded by the app, and your bookmark data is not sent to us.

Need support anyway?

Most users should need minimal support, but if you hit a blocker with setup or protected media access, send us an email and we will help you quickly.
