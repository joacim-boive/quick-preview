Feature overview

Focused tools for repeat review.

QuickPreview is designed around the moments that matter most when you are inspecting timing, movement, detail, and sensitive reference points inside a clip.

A compact interface centered on quick navigation, exact replay, and repeat review.

Playback

Play or pause instantly

QuickPreview keeps the core transport interaction immediate so repeated stop-start review feels natural instead of sluggish.

Precision

Fine seek, coarse seek, and frame-step

Move in small 0.1 second increments, jump a full second when needed, or use frame-step behavior to inspect exact points in a clip with more confidence.

Looping

Full-file or selected-segment replay

Toggle a full video loop or mark a narrower range to repeat just the section you are studying. It is built for comparison and repeated visual checks.

Workflow

Finder-aware clip switching

Once a video is loaded, selecting another file in Finder can switch playback to the newly selected clip while the player remains active.

Bookmarks

Bookmark manager for exact recall

Save precise moments, reopen them quickly, and manage your review points in a dedicated browser instead of relying on memory or rough timestamps.

Security

Dual-layer protection for private material

Protected bookmarks can stay behind macOS device-owner authentication and an optional paranoid-mode password for a stronger two-step unlock flow.

Designed for efficiency

Core workflows, not clutter.

The feature set is intentionally pointed at playback review, moment capture, and controlled access instead of general editing work, which keeps the interface faster to understand and faster to use.

High-resolution scrubber feel

Scrubbing is meant to help you target specific moments quickly and then refine with keyboard controls for precision.

Selection controls for loop ranges

Dedicated start and end markers make it easy to isolate a replay window and clear it when you are done.

Session-friendly state retention

The broader product direction includes remembering useful clip-level settings so repeat work becomes smoother over time.

Searchable bookmark library

The bookmark manager is built to help you sort through saved moments across clips, with support for searching, tagging, and switching between different bookmark views.

Protected bookmark visibility

Private bookmarks stay hidden until protected media is unlocked, making it easier to keep sensitive review points out of casual view.

Paranoid mode password step

When enabled, paranoid mode adds an extra password after macOS authentication, giving you a second layer before protected bookmarks appear.

Bookmarks

Your review moments, fully organized.

The bookmark manager is built for high-volume review work: import clips, locate moments instantly, and keep sensitive references protected.

Live bookmark manager view with searchable columns, tags, and private-state controls.

Drag-and-drop plus one-click imports

Bring in media by dropping files directly into the bookmark manager or using the Import button when you want a structured batch flow.

Zoomable native-resolution frame previews

Hover previews can open into larger, high-detail frame views so you can inspect image clarity and composition before jumping back to playback.

Powerful sorting and scoped views

Sort by time, filename, imported date, or file-created date, and switch quickly between Current Video, All Videos, and Imported scopes.

Fast search across key fields

Search by tags, timestamps, or filenames to locate a saved moment quickly when libraries grow.

Tagging for reusable review context

Add custom tags to bookmark rows to group themes, shots, or quality notes and retrieve them later without manual re-scrubbing.

Private states for sensitive references

Mark bookmarks as private and keep them hidden until protected media is unlocked, with optional paranoid-mode password for an extra layer.

Reference controls

Keyboard-first when speed matters.

Shortcut Action Notes
Esc Close the current preview window Fast exit when you are done reviewing a clip.
Space Play / pause Works as the primary transport toggle.
L Toggle loop on / off Applies to the current clip context.
R Rotate video clockwise Cycles through 0, 90, 180, and 270 degrees.
Left / Right Fine seek Adjusts by small steps (0.1s) for precision.
Shift + Left / Shift + Right Coarse seek Jumps by 1.0s when you need faster navigation.
Up / Down Frame step forward / backward Useful for frame-level inspection and timing checks.

Loop range controls are also available in the UI with Set Start, Set End, Replay Selection, and Clear Selection.
