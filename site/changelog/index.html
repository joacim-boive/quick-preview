Changelog

Release history.

A simple version history for QuickPreview. The latest tag browser and dynamic filtering improvements are grouped under 2.4, followed by the earlier 2.3, 2.2, and 2.1 feature releases.

Version 2.4

Tag browser and filtering upgrades

  • Added a dedicated Tags view in the bookmarks window with a tag cloud that shows clip counts for each visible tag.
  • Tag filtering now supports selecting and deselecting multiple tags with instant updates to matching clips and totals.
  • Tag cloud suggestions and counts now update dynamically based on the currently selected tag combination.
  • Protected media tags remain hidden while locked and appear only after protected media is unlocked.
Version 2.3

Launch restoration improvements

  • QuickPreview now restores the last viewed clip directly into the player when the app launches.
  • Protected clips stay blocked at launch until protected media is unlocked, so private content does not reopen unexpectedly.
  • The bookmarks window can now reopen automatically on launch if it was still open when the app last quit.
  • Bookmarks window size and screen position are now restored between launches for a more consistent workspace.
Version 2.2

Import and playback workflow polish

  • Imported media now handles duplicates with replace, skip, and batch apply options.
  • Duplicate imports can reveal the existing bookmark so conflicts are easier to review before deciding.
  • Bookmark row actions now include a quick way to reveal the source clip in Finder.
  • New autoplay toggle controls are available from both the player UI and the Playback menu.
  • Guide and playback polish improve the overall review flow and reduce setup friction.
Version 2.1

Bookmark workflow upgrades

  • Manual bookmark thumbnail selection after bookmark creation.
  • New per-bookmark actions menu with room for future tools.
  • Dedicated quick-remove trash action remains outside the menu.
  • Embedded video-based frame picker for smoother scrubbing and faster frame targeting.
Version 2.0

Major additions

  • Bookmark manager for saved moments across clips.
  • Up/Down arrows stay tied to bookmark browsing, while Shift + Up/Down handles frame stepping.
  • Protected bookmarks with optional dual-layer paranoid mode.
  • Per-clip state recall for position, loop preference, and context.
  • Search, sorting, tags, and imported-media bookmark views.
  • Quick close with Esc and improved repeat-review flow.
Version 1.5

Playback workflow improvements

  • Selected-range replay workflow with Set Start and Set End.
  • Fine seek, coarse seek, and precision frame-step controls.
  • Loop toggling and higher-confidence timing review.
  • Rotation and audio boost support.
Version 1.2

Session quality-of-life

  • Finder-aware clip switching while the player stays active.
  • Per-clip playback and loop preference persistence.
  • More polished shortcuts and repeated-review behavior.
Version 1.0

Initial release

  • Standalone macOS video player.
  • Fast open-and-review workflow.
  • Play, pause, scrub, and basic loop-focused playback.
