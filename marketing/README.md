# Website marketing package

This directory is the source of truth for QuickPreview's public product metadata, feature copy, changelog, icon, social preview, and screenshots.

The Boive portal publishes this material at:

- `https://boive.se/quick-preview/`
- `https://boive.se/quick-preview/features/`
- `https://boive.se/quick-preview/changelog/`

Keep private implementation details, credentials, build artifacts, and unpublished roadmap items outside this directory. Changes here will eventually open an automated content-update pull request in the portal repository.

The QuickPreview subscriber bridge remains in this repository and continues to run from its dedicated Vercel project.
