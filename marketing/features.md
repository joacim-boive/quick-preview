# QuickPreview features

QuickPreview is a focused macOS video player for people who need to move through clips quickly, keep important moments organized, return to the same context later, and protect anything that should stay private.

## Fast review

- Open clips from the file picker or with drag and drop.
- Use responsive scrubbing, 0.1-second fine seek, and 1-second coarse seek.
- Step through frames with Shift + Up/Down.
- Loop the whole clip or mark a smaller replay range.
- Rotate awkward footage and boost quiet audio.
- QuickPreview PRO adds Finder live-follow for active subscribers.

## Bookmark manager

- Save exact moments with thumbnail previews.
- Search across filenames, timestamps, and tags.
- Sort by time, filename, import date, or file creation date.
- Switch between current-video, all-video, and imported-media views.
- Import supporting media directly into the bookmark manager.
- Keep protected media tags hidden while protected content is locked.

## Remembered clip state

- Return to the same playback position.
- Restore window placement and size.
- Preserve time ranges, loop preference, rotation, volume boost, and other clip-level adjustments.

## Privacy

- Hide protected bookmarks until they are unlocked.
- Use macOS device-owner authentication as the first gate.
- Optionally require a second password with paranoid mode.
- Keep videos, bookmarks, and clip state local to the Mac.
