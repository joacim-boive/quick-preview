# QuickPreview changelog

## 2.4 — Tag browser and filtering upgrades

- Added a dedicated Tags view with a tag cloud and visible clip counts.
- Added multi-tag selection with instant updates to matching clips and totals.
- Made tag suggestions and counts respond to the current tag combination.
- Kept protected-media tags hidden until protected media is unlocked.

## 2.3 — Launch restoration improvements

- Restore the last viewed clip directly into the player at launch.
- Keep protected clips blocked at launch until protected media is unlocked.
- Reopen the bookmarks window automatically when it was open at quit.
- Restore the bookmarks window's size and screen position.

## 2.2 — Import and playback workflow polish

- Handle duplicate imports with replace, skip, and batch-apply options.
- Reveal existing bookmarks when reviewing duplicate conflicts.
- Reveal a bookmark's source clip in Finder.
- Control autoplay from the player UI and Playback menu.
- Improve the guide and general playback flow.

## 2.1 — Bookmark workflow upgrades

- Select bookmark thumbnails manually after bookmark creation.
- Use a per-bookmark actions menu.
- Keep quick removal available as a dedicated trash action.
- Pick frames with an embedded video-based scrubber.

## 2.0 — Major additions

- Added the bookmark manager.
- Added bookmark navigation and frame stepping.
- Added protected bookmarks and optional dual-layer paranoid mode.
- Added per-clip state recall.
- Added search, sorting, tags, and imported-media views.
- Added quick close with Esc and improved repeat-review workflows.

## 1.5 — Playback workflow improvements

- Added selected-range replay with start and end markers.
- Added fine seek, coarse seek, and precision frame stepping.
- Added loop controls, rotation, and audio boost.

## 1.2 — Session quality of life

- Added Finder-aware clip switching while the player remains active.
- Added per-clip playback and loop-preference persistence.
- Improved shortcuts and repeated-review behavior.

## 1.0 — Initial release

- Released the standalone macOS video player.
- Added fast open-and-review workflows.
- Added play, pause, scrubbing, and loop-focused playback.
